{{ 'dataProtection.usePersonalData' | translate }}

Aktiviere die Einstellung, um deine persönlichen Daten wie Name, Geburtsdatum oder Wohnort für die Infektinsauswertung und Kontaktaufnahme freizugeben.

{{ 'dataProtection.useGps' | translate }}

Damit aktivierst du die Aufzeichung von GPS Daten auf deinem Gerät. Durch die Aktivierung werden noch keine Daten versendet.

{{ 'dataProtection.useBluetooth' | translate }}

Damit aktivierst du die Aufzeichung von Bluetooth Kontakten mit anderen Personen auf deinem Gerät. Durch die Aktivierung werden noch keine Daten versendet.

{{ 'dataProtection.sendData' | translate }}

Aktiviere die Einstellung, um die gesammelten Daten zur Berechnung einer Infektionswahrscheinlichkeit an ein Rechenzentrum übermittelt. Wenn du diese Einstellung nicht aktivierst kannst du keine Infektionswarnung erhalten.
